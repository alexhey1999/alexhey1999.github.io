Welcome to Dead By Daylight Data Project

Open Source Project for gathering data.

About This API

Since the start of the game Behavior has always kept certain statistics from the game private. This project aims to change that. This API Tracks evry public match of Dead By Daylight that I play automatially to track the following things:

  • Perks
  • Killers
  • Escapes/Kills
  • Match Scores
  • Items (Beta)
  • Offerings (Beta)
  • Addons (Not Currently Implemented)

How It Works

These statistics are generated automatically using End Game Screenshots detailing every detail of match information. Currently, this seems to be the only way of tracking this data without breaking Behaviors TOS.

Using image processing in Python 3.9 we can analyze pixel clusters and determine what data is being used. This method avoids any internal tampering with the game and instead only uses.
